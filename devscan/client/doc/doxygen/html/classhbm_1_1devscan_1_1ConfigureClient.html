sends a request to multicast group CONFIG_IPV4_ADDRESS and waits some time (TIMETOWAITFORANSWERS) for the answer

Parameters
outgoingInterfaceIpIP address of the interface to use leave empty ("") in order to send over all interfaces
ttlnumber of maximum hops to the receiver (1 do not scan behind network routers)
ida (unique) string identifying the request to send and match the corresponding response
messagecomplete JSOM-message to send, as a plain string
Returns
empty string if no answer was received. Otherwise JSON rpc response from device.
